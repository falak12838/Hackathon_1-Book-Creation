# Module 1: The Robotic Nervous System (ROS 2)

This module covers the fundamentals of ROS 2 as the middleware nervous system for humanoid robots. It includes core communication concepts and robot description using URDF.

## Chapters

1. [Introduction to ROS 2 for Physical AI](./introduction-to-ros2-for-physical-ai.md)
2. [ROS 2 Communication Model](./ros2-communication-model.md)
3. [Robot Structure with URDF](./robot-structure-with-urdf.md)

## Learning Objectives

After completing this module, you will understand:
- What ROS 2 is and why it matters for humanoid robotics
- The OOS (Open, Operate, Share) concepts in collaborative robotics
- ROS 2 communication architecture (nodes, topics, services)
- How to create and validate URDF files for humanoid robots
- How to implement basic agent-controller communication patterns
