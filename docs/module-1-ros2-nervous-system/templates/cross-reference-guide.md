# Cross-Reference Guide

## Internal Links

To link to other pages within this module:

```markdown
[Link text](./relative-path-to-page)
```

Example:
```markdown
[Learn about ROS 2 Communication](./ros2-communication-model.md)
```

## Docusaurus Doc Links

To link to other documentation pages:

```markdown
[Link text](/docs/path-to-doc)
```

Example:
```markdown
[Learn about URDF](/docs/module-1-ros2-nervous-system/robot-structure-with-urdf)
```

## Reference Format

When referencing concepts from other chapters, use this format:
- "As discussed in [chapter name], [brief context]"
- "For more details on [topic], see [chapter name]"
